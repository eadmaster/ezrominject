## Patch instructions  🩹

1. Obtain a disc dump matching [these hashes](http://redump.org/disc/59761/)
2. Download the latest xdelta patch in this folder
3. Visit [Rom Patcher JS](https://www.marcrobledo.com/RomPatcher.js/)
4. Select `Rakugaki Showtime (Japan).bin` as ROM file
5. Select `.xdelta` as Patch file
6. Click "Apply patch" and save in the same folder as `"Rakugaki Showtime (English).bin`
7. Download and use the cue sheet in this folder to play the game
