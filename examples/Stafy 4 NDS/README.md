## Patch instructions  🩹

1. Visit [Rom Patcher JS](https://www.marcrobledo.com/RomPatcher.js/)
2. Download the latest xdelta patch in this folder
3. Select `Densetsu no Stafy 4 (Japan).nds` as ROM file (correct crc32: [45fe3a37](https://datomatic.no-intro.org/index.php?page=show_record&s=28&n=0401))
4. Select `Densetsu no Stafy 4 (English).nds.xdelta` as Patch file
5. Click "Apply patch"
