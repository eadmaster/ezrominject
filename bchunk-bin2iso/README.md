
This is a stripped-down verson of [binchunker](https://github.com/hessu/bchunk),
made to easily convert a single track from MODE1/2352 to MODE1/2048.

I made this because other similar tools like [iat](https://github.com/ropery/iat) refuse to produce any output
if the input does not contain a valid ISO9660 filesystem, and it is needed
to work with PC Engine CD disks without the ECC data getting in way.
